# dotfiles

Managed using [chezmoi](https://chezmoi.io). The `main` branch contains chezmoi source files (prefixed with `dot_`, `private_`, etc.). The [`rendered`](../../tree/rendered) branch contains the actual rendered dotfiles as they would be applied to a machine.

## Rendered branch

The `rendered` branch is auto-generated by `render_macos_output.sh` whenever changes are pushed from `main`. Template variables use placeholder values and `Library/` is excluded.

To render manually:

```sh
./render_macos_output.sh
```

## Git hook setup (one-time)

To have the rendered branch update automatically on every push:

```sh
git config core.hooksPath .githooks
```
